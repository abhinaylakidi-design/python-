Strings in Python

What is a String?

A string in Python is a sequence of characters enclosed inside single quotes (' '), double quotes (" "), or triple quotes (''' ''' or """ """).

name = "Python"
city = 'Hyderabad'
message = """Welcome to Python Programming"""

Creating Strings

  • Single quotes → 'Hello'
  • Double quotes → "Hello"
  • Triple quotes → used for multi-line strings
text1 = 'Hello'
text2 = "World"
text3 = '''This is
a multi-line
string'''

Accessing Characters (Indexing)

Each character in a string has an index. Indexing starts from 0.

word = "Python"
print(word[0]) # P
print(word[3]) # h

String Slicing

Slicing allows you to extract a portion of a string.

word = "Python"
print(word[0:4]) # Pyth
print(word[2:]) # thon

String Concatenation

Joining two or more strings is called concatenation.

first = "Hello"
second = "Python"
result = first + " " + second
print(result)

String Length

The len() function is used to find the length of a string.

name = "Programming"
print(len(name))

Common String Methods

  • upper() – converts to uppercase
  • lower() – converts to lowercase
  • strip() – removes spaces
  • replace() – replaces text
  • split() – splits string into list
text = " Python Programming "
print(text.upper())
print(text.strip())
print(text.replace("Python", "Java"))
print(text.split())

String Immutability

Strings in Python are immutable, meaning they cannot be changed after creation.

name = "Python"
name[0] = "J" # Error
✔ To modify a string, you must create a new one.

Conclusion

Strings are one of the most commonly used data types in Python. They are easy to use and come with powerful built-in methods that make text processing simple and efficient.
