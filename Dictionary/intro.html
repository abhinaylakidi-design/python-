Dictionaries in Python

1. What is a Dictionary?

A dictionary in Python is a built-in data structure used to store data in keyâ€“value pairs. Each key is used to access its corresponding value.

student = {
    "name": "Abhinay",
    "age": 22,

2. Characteristics of Dictionaries

  • Data is stored as key : value
  • Keys must be unique
  • Keys must be immutable (string, number, tuple)
  • Values can be of any data type
  • Dictionaries are mutable
  • Ordered from Python 3.7+

3. Accessing Dictionary Values

You can access values using keys.


print(student["name"])

Using get() method (safe way):


print(student.get("age"))
ðŸ‘‰ get() does not give an error if the key is missing.

4. Adding and Updating Values


student["college"] = "Sri Indu"
student["age"] = 23

Using update():


student.update({
    "city": "Hyderabad",
    "grade": "A"
})

5. Removing Items from Dictionary


student.pop("age")        # removes specific key
del student["course"]     # deletes key
student.clear()           # removes all items

6. Looping Through a Dictionary

Loop through keys:


for key in student:
    print(key)

Loop through values:


for value in student.values():
    print(value)

Loop through key-value pairs:


for key, value in student.items():
    print(key, ":", value)

7. Nested Dictionary

A dictionary inside another dictionary is called a nested dictionary.


employees = {
    "emp1": {"name": "Raj", "salary": 50000},
    "emp2": {"name": "Sai", "salary": 60000}
}

print(employees["emp1"]["name"])

8. Dictionary Comprehension


squares = {x: x*x for x in range(1, 6)}
print(squares)

9. Real-Time Example


login = {
    "username": "admin",
    "password": "1234"
}

if login["username"] == "admin":
    print("Login Successful")

10. Summary

  • Dictionaries store data in key-value format
  • Fast data access
  • Widely used in real-world applications
  • Very important for interviews and projects
