For Loop in Python

What is a For Loop?

A for loop in Python is used to repeat a block of code for each item in a sequence such as a list, string, or range of numbers.

Syntax of For Loop

for variable in sequence: statement

Python uses indentation to define the block of code inside the loop.

Example 1: Print Numbers from 1 to 5

for i in range(1, 6): print(i)
Output: 1 2 3 4 5

How It Works

  • range(1, 6) generates numbers from 1 to 5
  • The loop runs once for each number
  • i stores the current value

Example 2: Loop Through a List

fruits = ["Apple", "Banana", "Mango"] for fruit in fruits: print(fruit)
Output: Apple Banana Mango

Example 3: Print Even Numbers

for i in range(2, 11, 2): print(i)
Output: 2 4 6 8 10

Example 4: For Loop with Condition

for i in range(1, 11): if i % 2 == 0: print(i, "is even")

Important Points to Remember

  • Used when number of iterations is known
  • Works with lists, strings, tuples, and ranges
  • Indentation is mandatory in Python
  • Cleaner and simpler than many other loops

Conclusion

The for loop is one of the most commonly used loops in Python. It helps in repeating tasks efficiently and makes code easy to read and understand.
