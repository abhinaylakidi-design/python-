While Loop in Python

What is a While Loop?

A while loop in Python is used to execute a block of code repeatedly as long as a given condition is true.

Syntax of While Loop

while condition: statement

The loop continues running until the condition becomes false. Proper indentation is mandatory in Python.

Example 1: Print Numbers from 1 to 5

i = 1 while i <= 5: print(i) i += 1
Output: 1 2 3 4 5

How It Works

  • i = 1 initializes the variable
  • Condition i <= 5 is checked
  • i += 1 increments the value
  • Loop stops when condition becomes false

Example 2: Print Even Numbers

i = 2 while i <= 10: print(i) i += 2
Output: 2 4 6 8 10

Example 3: Countdown Using While Loop

i = 5 while i >= 1: print(i) i -= 1
Output: 5 4 3 2 1

Example 4: While Loop with Condition

i = 1 while i <= 10: if i % 2 == 0: print(i, "is even") i += 1

Important Note: Infinite Loop

If the condition never becomes false, the loop will run forever.

while True: print("This is an infinite loop")

Important Points to Remember

  • Used when number of iterations is not known
  • Condition is checked before every iteration
  • Loop variable must be updated manually
  • Wrong condition can cause infinite loops

Conclusion

The while loop is useful when a task must continue until a condition is met. It provides flexibility but requires careful handling of the loop condition.
