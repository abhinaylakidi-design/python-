Regular Expressions in Python

1. What are Regular Expressions?

Regular Expressions (Regex) are patterns used to match, search, extract, and replace text. In Python, regular expressions are handled using the built-in re module.

Uses of Regex:
  • Input validation
  • Pattern searching
  • Text extraction
  • Data cleaning

2. Importing re Module

import re

3. Regex Functions in Python

Function Description
re.match() Matches pattern at the beginning
re.search() Searches pattern anywhere in string
re.findall() Returns all matches as a list
re.finditer() Returns iterator of match objects
re.sub() Replaces matched text
re.split() Splits string using pattern

4. re.match()

import re
text = "Python Programming"
result = re.match("Python", text)
print(result)

5. re.search()

result = re.search("Programming", text)
print(result.group())

6. re.findall()

text = "apple banana apple orange"
result = re.findall("apple", text)
print(result)

7. re.finditer()

for match in re.finditer("apple", text):
    print(match.start(), match.group())

8. re.sub()

text = "I like Java"
result = re.sub("Java", "Python", text)
print(result)

9. re.split()

text = "one,two;three four"
result = re.split("[,; ]", text)
print(result)

12. Validation Examples

Email Validation

email = "test@gmail.com"
pattern = r"^[a-zA-Z0-9._]+@[a-zA-Z]+\.[a-zA-Z]{2,}$"

if re.match(pattern, email):
    print("Valid Email")

Mobile Number Validation

mobile = "9876543210"
pattern = r"^[6-9]\d{9}$"

if re.match(pattern, mobile):
    print("Valid Mobile Number")

13. Greedy vs Non-Greedy

text = "
Title
"

print(re.findall("<.*>", text))    # Greedy
print(re.findall("<.*?>", text))   # Non-Greedy
